# Fylo Data Storage Component

Solución al reto Fylo data storage component https://www.frontendmentor.io/challenges/fylo-data-storage-component-1dZPRbV5n de Frontend Mentor.

## Construido con

- HTML5
- CSS3

## Lo que aprendí

Practiqué cómo crear una barra de progreso personalizada con `linear-gradient` y un círculo al final usando `::after` sin elementos extra en el HTML. También usé `position: absolute` con un triángulo CSS en `::after` para crear la "cola" del tooltip de almacenamiento en desktop.
